# Claude Code Marketplace

A personal marketplace for Claude Code plugins, commands, skills, and agents.

## Quick Start

### Use local marketplaces for development

Test your marketplace locally before distribution:

```bash
/plugin marketplace add ~/claude-code-marketplace
```

## Available Plugins

- **[GitHub Plugin](plugins/github/README.md)** - AI-powered GitHub workflow automation
